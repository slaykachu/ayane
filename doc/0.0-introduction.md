Penultimate (second to last) AY-3-8910 tracker for Amstrad CPC 6128.

# Quick start

* RUN"ayane"
* Some instruments (not named yet!) are pre-loaded
* Test them via the instrument selector (CTRL-I, then up/down and piano keys)
* You can start composing in the phrase editor (CTRL-f2). See [implicit sequencing](./1.0-sequencer.md#implicit-sequencing)
* Load VAMPSYNC.AYA (CTRL-O) for a demo song with dual Sync Buzzer 
* CTRL-H for HELP


# Features, in a nutshell (short list, alpha version !!):

* Timer FXs: Just Multi Sync Buzzer for now 
* Free duration notes (delay agnostic)
* 99 instruments (limit could be expended)
* 255 phrases
* No Acorn/Amiga/Atari/PC needed
